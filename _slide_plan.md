## 12. 자동화 필요성 (why automation)
- 학교 업무 특징: 공문 본문+첨부 동시 확인, 작년·올해 비교, 표·명단·예산 반복 정리, 결과 다시 문서로 저장
- AI가 도울 부분: 초안 작성, 자료 요약·비교, 표 정리·누락 확인, 파일명·폴더 구조 제안
- 메시지: "AI를 답변기에서 파일 작업 도우미로 확장"

## 13. 웹 UI vs 데스크톱 앱 (web vs desktop)
표 (블로그 인용):
| 구분 | 웹 UI | 데스크톱·로컬 AI |
| 시작 | 브라우저 | 앱·CLI 설치 |
| 파일 | 업로드·붙여넣기 | 허용 폴더 직접 작업 |
| 흐름 | 단발 질문·답변 | 다단계 작업 |
| 결과 | 사용자가 복사 | 새 파일 저장·수정·변환 |
| 강점 | 문장·아이디어 | 반복 업무·파일 정리 |
- 메시지: "웹 UI는 유지, 파일 기반 업무는 데스크톱으로 확장"
- 이미지: 08-web-vs-desktop.png (채팅창 ↔ 폴더 작업대)

## 14. 기본 도구 3종 (tools)

- ③ Codex: 파일 수정·변환·명령 실행 작업형 에이전트 → 반복 작업·파일 변환·산출물 생성
- 메시지: "질문 종류가 아니라 업무 형태로 도구 선택"
